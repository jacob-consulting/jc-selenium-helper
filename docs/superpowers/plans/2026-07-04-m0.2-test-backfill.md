# M0.2 browser.py Test Backfill Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add behavior-asserting tests for the untested `browser.py` methods and failure paths, raising `browser.py` coverage from 46% toward ~85-90%.

**Architecture:** Purely additive tests + HTML fixtures. No production code changes. Tests slot into the existing per-concern test files and reuse the existing `browser` + `fixture_url` fixtures. One new fixture (`dynamic.html`) supplies reactive elements (disappearing element, hover recorder, file input); `inner.html` gains an Enter-key recorder so `fill_in_frame(submit=True)` is observable.

**Tech Stack:** Python ≥3.12, pytest, selenium (live headless Chrome), local `file://` HTML fixtures.

## Global Constraints

- **No changes under `src/`** — this is a test-only backfill. If a test seems to require a code change, stop and flag it.
- **No changes to `tests/conftest.py`** — reuse the existing `browser` (`default_timeout=10`, `poll_pause=0.2`) and `fixture_url` fixtures.
- Tests must pass on Python 3.12 / 3.13 / 3.14 with live headless Chrome (matches CI).
- `ruff check src tests` and `ruff format --check src tests` must stay clean.
- These are backfill/characterization tests: production code already exists, so a correctly-written test **passes** on first run. Each assertion must verify real behavior (a value/text/exception), never merely "did not raise".
- `hover_with_offset` and the `wait_document_ready` timeout branch are **out of scope** (deferred to M3.2 / not practically testable over `file://`).

- [ ] **Step 2: Add the submit-path test**

```python
def test_fill_in_frame_submit_sends_return(browser, fixture_url):
    browser.open(fixture_url("frame.html"))
    browser.fill_in_frame(
        "//iframe[@id='editor']",
        "//input[@id='inner-input']",
        "typed",
        submit=True,
    )
    browser.driver.switch_to.frame(browser.find("//iframe[@id='editor']"))
    assert browser.find("//div[@id='submitted']").text == "typed"
    browser.driver.switch_to.default_content()
```
